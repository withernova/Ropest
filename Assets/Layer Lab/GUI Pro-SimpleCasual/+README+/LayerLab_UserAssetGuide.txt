##Bundle Pack User Guide

With a Bundle Pack, you can purchase multiple connected assets at a discounted price.
Once bought, you'll have free access to all assets connected to the pack.
You can download the assets individually for free through the Package Manager.
For more details, please refer to the "GUI Bundle Pack" menu via the link above.
